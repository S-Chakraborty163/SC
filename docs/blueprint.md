# **App Name**: Soumyajit's ByteFolio

## Core Features:

- Dynamic Hero Section: Displays an engaging 'Hero' landing area with personalized greetings, quick navigation buttons to 'Projects' and 'Contact', and a reference background image.
- Comprehensive 'About Me' Display: Showcases personal introduction, research interests, detailed experience, and skills list in a clean, card-based format with a subtle background.
- Interactive Project Portfolio: Presents academic and personal projects using an appealing card-based layout, allowing easy browsing and access to project details.
- AI-Powered Learning Insights Tool: Leverages generative AI to provide concise, on-demand summaries or key takeaways from listed 'Documented Learnings' GitHub repositories, aiding quick comprehension.
- Documented Learnings Showcase: Lists external links to detailed learning repositories on platforms like GitHub, categorized for easy exploration.
- Contact and Resume Hub: Integrates essential contact information (email, LinkedIn, GitHub), a functional contact form, and a dedicated resume download button.
- Personal Interests Showcase: Highlights personal hobbies and extracurricular activities such as football, photography, and trekking with accompanying imagery for visual context.

## Style Guidelines:

- A dark mode theme that conveys professionalism and depth. The primary action color, used for interactive elements and highlights, is a vibrant light royal blue (#7792F0). The background is a very dark, desaturated blue-gray (#1D1F27), providing a calm and focused visual environment. An accent color of medium purple (#9E5AFF) is used sparingly for special emphasis and secondary interactive states, evoking creativity and insight.
- The font 'Inter' (sans-serif) is recommended for both headlines and body text due to its modern, neutral, and highly readable characteristics, aligning with a 'clean' aesthetic while maintaining objectivity for technical content.
- Use clean, minimalist line icons or glyphs that complement the dark mode theme, maintaining consistency across sections like skills, contact, and hobbies to avoid clutter.
- Implement a 'clean' and uncluttered layout with ample negative space to enhance readability. Projects and relevant information should be presented in card-based components for organized, digestible content presentation.
- Subtle animations, primarily 'hover effects', should be incorporated for interactive elements like buttons, cards, and links to provide visual feedback and a refined user experience without being flashy.
